# Upper Fibers and Witness Covers for the Fibonacci Order-of-Apparition Map

## Key Innovation

We introduce witness covers for the fiber B_n (divisors of F_n whose rank of apparition equals n), proving B_n is an upper set in the divisor lattice obtained by removing covering ideals at maximal proper divisors. Minimal generators of B_n are classified via a primitive-vs-ladder dichotomy for atomic prime-power birth moduli. Both B_n and its minimal-generator set admit unique factorizations into connected coordinate blocks, yielding partition-lattice formulas and Bell-type lower bounds.

**Scope:** Combinatorial number theory  
**Keywords:** Fibonacci sequences, rank of apparition, divisor lattices, multiplicative number theory
